# Supplementary Material S1: GDPO Manuscript Validation

This directory contains the article-specific fixture, queries, and OWL-RL checks
reported in *Principles Are Not Properties*. It uses the current GDPO v4.0.3
`owl:allValuesFrom` target-expression design and does not use punned target
individuals or `owl:hasValue` target restrictions.

## Contents

- `fixture.ttl` — the illustrative SK4/iPhone ABox from Appendix A1
- `queries/` — the six exercised competency-query templates plus the additional
  A2.3 artifact-comparison template
- `run_validation.py` — executable query, structural, and entailment checks
- `requirements.txt` — pinned RDFLib and OWL-RL versions
- `expected-output.txt` — expected deterministic report
- `validation-output.txt` — report regenerated by the runner
- `supplement.md` — narrative Supplementary Material S1 source

The product names and numeric scores are illustrative test data. They are not
empirical judgments about the named products.

## Run

From this directory:

```bash
python3 -m pip install -r requirements.txt
python3 run_validation.py
```

The runner exits nonzero if a query row count changes, either reported
entailment disappears, `owl:hasValue` returns to the ontology, a GDPO class is
also declared as a named individual, or the generated report differs from
`expected-output.txt`.

## Exercised queries

The paper reports four instance-data queries and two schema-inspection queries:

- Instance data: A2.2, A2.4, A2.5, and A2.6
- Schema inspection: A2.1 and A2.7

A2.3 is retained as an additional cross-artifact comparison template and is
also regression-tested, but it is not counted among the six competency-query
forms reported in the article.

1. `Eval_SK4_Durability_2024` has both the canonical durability statement
   (`GDPO0000072`) and canonical environmental-friendliness statement
   (`GDPO0000074`) as evaluation-relevant principles.
2. `Eval_SK4_ManualHonesty_2024` is classified as a design evaluation record
   (`GDPO0000044`), design honesty evaluation (`GDPO0000455`), and design
   communicative honesty evaluation (`GDPO0000452`).
