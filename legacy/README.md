# Legacy GDPO Ontology Files

This directory archives historical GDPO ontology files from the earlier public repository.

These files are retained for provenance and comparison only. They do not reflect the current argument, modeling commitments, import strategy, or paper-aligned ontology structure.

Current release:

```text
../ontology/gdpo-4.0.3.ttl
```

Legacy layout:

| Path | Contents |
|------|----------|
| `paper/` | Earlier manuscript drafts retained for provenance |
| `root-snapshots/` | Older root-level ontology snapshots from the previous repository state |
| `releases/` | Historical release folders copied from the previous repository |

Use the current release and current paper for new work. Treat files in this directory as archived historical material, not as active ontology sources.
